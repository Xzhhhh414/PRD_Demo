# review-prd

审核 PRD / 设计文档的完整性、可操作性和边界条件覆盖度。

## 调用方式

```
/prd-workflow:review-prd docs/xxx-prd.md
/prd-workflow:review-prd 功能关键词
```

## 执行流程概览

```
┌─ 定位文档 ─────────────────────────┐
│ 搜索匹配文档 → ★ 用户选择          │
└────────────────────────────────────┘
                  │
                  ▼
┌─ 读取文档 ─────────────────────────┐
│ Read 完整 PRD 内容                  │
└────────────────────────────────────┘
                  │
                  ▼
┌─ PRD 审核 ─────────────────────────┐
│ PRD Reviewer Agent 逐维度审查       │
│ 完整性 / 可操作性 / 边界 / 一致性   │
└────────────────────────────────────┘
                  │
                  ▼
┌─ 输出结果 ─────────────────────────┐
│ PASS → 进入设计阶段                 │
│ FAIL → 列出问题，回 Obsidian 修改   │
└────────────────────────────────────┘
                  │
                  ▼
┌─ 自优化 ──────────────────────────┐
│ ★ 提案 → 用户确认 → 更新 Agent    │
└────────────────────────────────────┘
```

## 产出物

- PRD 审核报告（PASS/FAIL + 问题清单）

## 涉及 Agent

- PRD Reviewer (`agents/prd-reviewer.md`)
